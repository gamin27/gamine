# Repository Instructions

Before making design, layout, animation, styling, or component-structure changes, read and follow [DESIGN.md](./DESIGN.md).

Key points:

- Use the current main color and background color as the design foundation.
- Avoid unnecessary animation.
- Use spacing based on multiples of `4px`.
- Keep the site fast, quiet, and deliberate.
- Extract a component when the same code appears in two or more places.
